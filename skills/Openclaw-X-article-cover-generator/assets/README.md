# Assets

Note: ClawHub currently may not package binary image files in some publish flows.

Place your primary reference image at:

- `assets/reference-main.jpg`

You can copy your local reference into this path before running generation.

Sample command:

```bash
cp /path/to/your/reference.jpg ~/.openclaw/workspace/skills/Openclaw-X-article-cover-generator/assets/reference-main.jpg
```
